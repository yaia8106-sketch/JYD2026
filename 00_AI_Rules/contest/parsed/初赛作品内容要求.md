# 九、"竞业达"企业命题 — 初赛作品内容要求

## 提交要求

| 提交文件夹 | 材料 | 内容要求 |
|-----------|------|--------|
| **技术文档** | 设计文档 | (1) CPU设计文档：按照模版提交，工程文件不需要提交，但需要提交报告中用到的bit文件，用于复核结果。（提交至技术数据文件夹） |
| | | (2) CPU测评报告：企业提供线上测评程序，学生需要按照操作要求实现测评结果的显示，完成至少37条指令，显示结果需要在数码管和LED上显示，线上提交截图和文档。 |
| **演示视频** | 演示讲解视频 | 视频要求10分钟以内，主要显示烧录的过程和结果。 |
| **技术数据（代码类）** | 工程资料及过程文档 | 报告中用到的bit文件，用于复核结果。 |

## 补充提醒

企业提供报告内容模板，参赛团队可参考内容规划，格式、排版等规则请按照大赛统一要求完成。模板下载见本文文末附件。

## 说明

1. 本文题目作品要求及评分标准可能较官网赛题中的要求有所差异，此次初赛环节以本文中的要求为准，如果在提交截止前命题企业对本通知内的要求有进一步补充，将会在其赛题钉钉群内提醒通知。

2. 参赛团队需按照给定要求将作品上传至对应的文件夹，其中所有题目的技术数据（如有）皆打包成压缩包格式提交，并遵守赛题针对此次作品给出的特别要求，其他关于格式或内容的未尽事宜请参考初赛通知附件2中的大赛统一格式要求，合理规划内容排版，以便评审专家更好的查阅作品。

3. 提交系统内的文件夹为统一预设模板，所有赛题皆一致，非内容提交要求（即如某赛题在赛题或本文中未要求提交PPT，则系统内的PPT文件夹中无需提交材料）。
